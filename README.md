# BrightTV Viewership Analytics Case Study

## Overview
This project analyzes BrightTV's user profiles and viewing transaction data to provide actionable insights for the Customer Value Management (CVM) team. The goal is to grow the company's subscription base by understanding user behavior, identifying factors that influence consumption, and developing targeted content and growth strategies.

## Key Deliverables
- SQL queries for data cleaning, transformation, and analysis
- Interactive Excel dashboards with pivot charts
- 20-minute presentation deck with strategic recommendations
- Project plan and mind map for portfolio documentation

## Tools Used
- **Databricks** (SQL) – Data preparation and analysis
- **Excel** – Visualization and chart creation
- **PowerPoint** – Final presentation

## Key Findings
- 76% of users are inactive – never watched any content
- Monday has 61% lower consumption than Saturday
- 5 PM SA time is optimal for push notifications (584 active users)
- Gauteng males aged 30+ are the primary inactive user segment
- Male users dominate platform (88% of active users)

## Recommendations
- Schedule Monday night premieres to boost low day consumption
- Reactivate 989 inactive users with targeted campaigns
- Send push notifications at 5 PM SA time for maximum reach
- Study Eastern Cape user behavior (highest engagement at 25.6 hrs/user)
